\section{統合設計フロー}
SystemDK with AITL Core の設計フローを以下に示す。
\begin{verbatim}
仕様策定 → FEM解析(熱/応力/ノイズ)
        → PID制御設計
        → FSM状態遷移定義
        → LLM整合検証
        → SystemDK統合
\end{verbatim}
各段階が同一スキーマで接続され，
信頼性と制御安定性の整合が保証される。
